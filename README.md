# Data Cleaning & Preparation Project

## Overview

This project was completed as part of my Data Analytics Internship with Decode Labs.

The objective of this project was to clean and prepare an e-commerce dataset using Microsoft Excel by:

* Identifying missing values
* Handling null data
* Checking duplicate records
* Validating data formats
* Creating a structured change log

---

## Tools Used

* Microsoft Excel

---

## Dataset Information

* Dataset Type: E-commerce Orders Dataset
* Total Records: 1200
* Total Columns: 14

---

## Data Cleaning Process

### 1. Missing Value Handling

* Identified missing values in the `CouponCode` column.
* Replaced blank entries with:

```text
No Coupon Used
```

### 2. Duplicate Check

* Verified dataset for duplicate records.
* No duplicate rows were found.

### 3. Data Validation

* Checked date formats
* Verified numerical columns
* Validated text consistency
